Week 1 Variables & Data Types

1
Variables & Print
📖 Description

Variables are containers for storing data. Python automatically detects the type. Print displays output.

🎓 Tutor

Variables store values. No need to declare type — Python figures it out:

name = "Olami" → string

age = 25 → integer

price = 19.99 → float

is_active = True → boolean

Print shows output. Use f-strings for formatted output:

print(f"Hello, {name}!")

✏️ Practice
  • Create a variable for your name
  • Create a variable for your age
  • Print both using f-string

2
User Input
📖 Description

input() gets data from users. Always returns string — convert with int() or float().

🎓 Tutor

Get input from users:

name = input("What's your name? ")

age = int(input("Your age: "))

price = float(input("Price: "))

✏️ Practice
  • Ask for user's name
  • Ask for their age
  • Print a greeting with both
⚡ Try It Now
3
String Formatting
📖 Description

f-strings format text cleanly. Concatenate with +, escape with \.

🎓 Tutor

f-strings (recommended):

f"Hello, {name}!"

Concatenation:

"Hello, " + name

Escape characters:

"Line 1\nLine 2" (new line)

"She said \"hi\"" (quotes)

✏️ Practice
  • Create variable with quotes in it
  • Print on two lines using \n
  • Use f-string to combine 3 variables
⚡ Try It Now
4
Math Operators
📖 Description

Python handles math: +, -, *, /, // (floor), % (modulo), ** (power).

🎓 Tutor

a + b → addition

a - b → subtraction

a * b → multiplication

a / b → division (decimal)

a // b → floor division (whole)

a % b → remainder (modulo)

a ** b → power (a^b)

✏️ Practice
  • Calculate 10 + 5, print result
  • Find remainder of 17 ÷ 5
  • Calculate 2 to the power of 10

Week 2 Conditions & Loops

1
If/Else Statements
📖 Description

if/else makes decisions. Code runs based on conditions being True or False.

🎓 Tutor

if age >= 18:

  print("Adult")

else:

  print("Minor")

elif for multiple conditions:

if score >= 90: print("A")

elif score >= 80: print("B")

else: print("C")

✏️ Practice
  • Check if number is positive/negative
  • Check if user can vote (age >= 18)
  • Use elif for A/B/C grades
2
Comparison Operators
📖 Description

Compare values: == (equal), != (not equal), <, >, <=, >=. Returns True/False.

🎓 Tutor

a == b → equal

a != b → not equal

a < b → less than

a > b → greater than

a <= b → less or equal

a >= b → greater or equal

✏️ Practice
  • Compare 5 == 5 (True?)
  • Compare "hello" != "world"
  • Check if 10 > 5 and 10 > 20
3
For Loops
📖 Description

for loops repeat code for each item in a sequence. range() generates numbers.

🎓 Tutor

Loop through range:

for i in range(5):

  print(i) → 0,1,2,3,4

range(start, end):

for i in range(1, 6):

  print(i) → 1,2,3,4,5

Loop through list:

fruits = ["apple", "banana"]

for fruit in fruits:

  print(fruit)

✏️ Practice
  • Print numbers 1 to 10
  • Print even numbers 2-20
  • Loop through list and print each
4
While Loops
📖 Description

while loops repeat while condition is True. Use break to exit, continue to skip.

🎓 Tutor

Basic while:

count = 0

while count < 5:

  print(count)

  count += 1

Break (exit loop):

while True:

  if input("quit? ") == "yes":

    break

✏️ Practice
  • Countdown from 5 to 1
  • While loop that runs until user types "stop"
  • Use break to exit infinite loop

Week 3 Data Structures

1
Lists
📖 Description

Lists hold multiple items. Ordered, mutable (changeable). Index starts at 0.

🎓 Tutor

Create list:

fruits = ["apple", "banana", "cherry"]

Access items:

fruits[0] → "apple"

fruits[-1] → "cherry" (last)

Modify:

fruits.append("orange")

fruits[0] = "mango"

fruits.remove("banana")

✏️ Practice
  • Create list of 5 colors
  • Print 2nd and 4th items
  • Add new color, remove one
2
Tuples & Sets
📖 Description

Tuples are immutable (can't change). Sets have unique values only.

🎓 Tutor

Tuple (immutable):

point = (10, 20)

x, y = point → unpacking

Set (unique values):

numbers = {1, 2, 3, 3, 3}

print(numbers) → {1, 2, 3}

numbers.add(4)

numbers.remove(2)

✏️ Practice
  • Create tuple with coordinates
  • Unpack tuple into x, y
  • Create set with duplicates, see result
3
Dictionaries
📖 Description

Dictionaries store key-value pairs. Fast lookup by key.

🎓 Tutor

Create dict:

person = {"name": "Olami", "age": 25}

Access:

person["name"] → "Olami"

person.get("age") → 25

Modify:

person["age"] = 26

person["city"] = "NYC"

del person["age"]

✏️ Practice
  • Create dict with name, email, phone
  • Print each value
  • Add new key, update existing
4
List Comprehension
📖 Description

List comprehension creates lists in one line. Compact and Pythonic.

🎓 Tutor

Basic:

squares = [x**2 for x in range(5)]

# [0, 1, 4, 9, 16]

With condition:

evens = [x for x in range(10) if x % 2 == 0]

# [0, 2, 4, 6, 8]

✏️ Practice
  • Create list of cubes 1-5
  • Filter list to get only negatives
  • Double each number in list

Week 4 Functions & OOP

1
Functions
📖 Description

Functions are reusable code blocks. Define with def, call by name.

🎓 Tutor

Define function:

def greet(name):

  return f"Hello, {name}!"

Call it:

greet("Olami")

Default parameter:

def greet(name="World"):

✏️ Practice
  • Create function add(a, b)
  • Function with default param
  • Function that returns multiple values
2
Args & Kwargs
📖 Description

*args accepts multiple positional arguments. **kwargs accepts keyword arguments.

🎓 Tutor

*args (tuple):

def sum_all(*args):

  return sum(args)

sum_all(1, 2, 3)

**kwargs (dict):

def info(**kwargs):

  for k, v in kwargs.items():

info(name="Olami", age=25)

✏️ Practice
  • Function that accepts any number of numbers
  • Function that prints all kwargs
  • Combine *args and **kwargs
3
Intro to OOP
📖 Description

Classes define blueprints for objects. Objects have attributes (data) and methods (functions).

🎓 Tutor

Create class:

class Person:

  def __init__(self, name, age):

    self.name = name

    self.age = age

Create object:

p = Person("Olami", 25)

print(p.name)

✏️ Practice
  • Create Car class with brand, color
  • Create object and print attributes
  • Add a method to the class
4
Class Methods
📖 Description

@staticmethod doesn't need self. @classmethod gets class itself as first param.

🎓 Tutor

Instance method:

def greet(self):

  return f"Hi, {self.name}"

Static method (no access to class):

@staticmethod

def add(a, b): return a + b

Class method:

@classmethod

def from_dict(cls, d):

  return cls(**d)

✏️ Practice
  • Add static method to calculate area
  • Add class method to create from dict
  • Call both types of methods
